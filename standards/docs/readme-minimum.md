# README Minimum Standard

- Status: Active
- Version: 1.1.0
- Source ADRs: ADR-0001, ADR-0002

## Required Sections

- Project name and short description
- Scope / non-goals
- Local setup and common commands
- Testing notes
- Contribution guidance link
- License section (or explicit internal-use note)

## Adoption Rule (v1.1.0)

If repository `README.md` already exists, preserve it by default during sync and treat this standard as a checklist for manual alignment.
