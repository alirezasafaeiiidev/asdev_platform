# Brand Template Pack v1

## Files

- `src/lib/brand.ts`
- `src/components/brand/BrandFooter.tsx`
- `src/components/brand/BrandLink.tsx`

## Usage

1. Copy files into target repository.
2. Install `BrandFooter` in root layout/footer.
3. Add `BrandLink` in header/help menu.
4. Verify engineering URLs match canonical standard.
